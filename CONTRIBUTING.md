# Contributing

Bug reports, security reports, feature requests, documentation corrections, and reproducible test cases are welcome.

Draftill is source-available and does not grant permission to modify the first-party code. Do not submit code patches or pull requests unless a maintainer has explicitly invited the contribution and a separate written contribution agreement is in place.

When reporting a bug:

- use the latest release;
- provide minimal reproduction steps;
- describe expected and actual behavior;
- include relevant screenshots when safe; and
- remove screenplay content, API keys, personal paths, and other sensitive information.
