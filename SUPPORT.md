# Support

Use GitHub Issues for reproducible bugs and feature requests.

Before opening an issue:

1. Confirm you are using the latest official release.
2. Remove API keys, screenplay content, file paths containing personal information, and other private data from screenshots or logs.
3. Include Windows version, Draftill version, steps to reproduce, expected behavior, and actual behavior.

This repository does not promise service-level response times. Source-code modification support is not provided under the public license.
