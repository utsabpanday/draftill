# Security policy

## Supported version

Security fixes currently target the latest published Draftill release.

| Version | Supported |
| --- | --- |
| 0.4.15 | Yes |
| Older versions | No |

## Reporting a vulnerability

Do not post API keys, private screenplay files, personal information, or exploit details in a public issue.

Use GitHub's private vulnerability reporting feature when it is enabled for the repository. Otherwise, contact the repository owner privately and include:

- the affected version;
- operating system and architecture;
- reproduction steps;
- impact assessment; and
- any suggested mitigation.

Please allow reasonable time for investigation before public disclosure.
